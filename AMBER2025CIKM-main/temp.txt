不同数据集训练命令（AMBER）

一、通用说明
- 请在项目根目录执行命令：AMBER2025CIKM-main
- 每个数据集都要先在 KnowledgeTracing/Constant/Constants.py 里切换 DATASET / NUM_OF_QUESTIONS / H
- 两阶段训练：先 teacher，再 student
  1) python -m KnowledgeTracing.run.run_tkt
  2) python -m KnowledgeTracing.run.run_mns


二、ASSIST2009
1) 修改 Constants.py
- DATASET = datasets['assist2009']
- NUM_OF_QUESTIONS = numbers['assist2009']
- H = '2009'

2) 训练命令
python -m KnowledgeTracing.run.run_tkt
python -m KnowledgeTracing.run.run_mns

四、Statics2011
1) 先在 Constants.py 增加数据集注册（若未加）
- datasets 增加: 'statics2011': 'statics2011'
- numbers 增加:  'statics2011': <题目总数>

2) 修改 Constants.py
- DATASET = datasets['statics2011']
- NUM_OF_QUESTIONS = numbers['statics2011']
- H = '2011'

3) 训练命令
python -m KnowledgeTracing.run.run_tkt

六、只训练单阶段（可选）
如果只想训练 teacher，不做 AMBER 蒸馏：

七、输出文件位置
- teacher: KnowledgeTracing/model/save<H>modelS_weights_teacher.pth
- student: KnowledgeTracing/model/save<H>modelS_weights.pth
- 日志: KnowledgeTracing/run/log/
